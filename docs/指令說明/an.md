# /an 公告代發

## 功能概述
/an 指令用於代為發佈正式公告，確保重要訊息以一致且清楚的格式呈現。

## 使用資格
- 僅限銀行公告頻道使用。
- 需具備銀行經理（含）以上身分組。

## 功能說明
輸入 `/an` 後會開啟輸入面板。填寫公告標題與內容後，系統即會建立正式公告訊息。若需附加圖片，可提供直接可讀取的連結，常見的圖床服務如下：
- https://duk.tw/
- https://urusai.cc/

系統允許透過文字內容使用 `@everyone` 與 `@here` 標記，但建議僅在必要情況下使用。

## 操作流程
1. 輸入 `/an` 並送出。
2. 填寫公告內容。
3. 提交後系統會自動於指定頻道發布公告。
